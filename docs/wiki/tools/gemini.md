# Gemini

!!! info "최종 수정일: 2025-01-15"

Google이 개발한 멀티모달 AI입니다. Google 생태계와의 통합과 초대용량 컨텍스트가 특징입니다.

| 플랜 | 가격 | 주요 기능 |

| **Free** | 무료 | Gemini 1.5 Flash, 기본 기능 |

## 모델 라인업

### Gemini 2.0 Flash

2024년 12월 출시된 최신 모델입니다.

**특징**:

- 이전 버전 대비 2배 빠른 속도
- 에이전트 기능 강화
- 멀티모달 출력 (이미지, 오디오)
- 향상된 코딩 능력

### Gemini 1.5 Pro

강력한 성능의 프리미엄 모델입니다.

**특징**:

- 2M 토큰 컨텍스트 (약 5,000페이지)
- 복잡한 추론 작업에 적합
- 장문 문서 분석에 최적

### Gemini 1.5 Flash

빠르고 효율적인 범용 모델입니다.

**특징**:

- 1M 토큰 컨텍스트
- 빠른 응답 속도
- 일상적인 작업에 최적

## 핵심 기능

### 1. 초대용량 컨텍스트

업계 최대인 2M 토큰 컨텍스트를 지원합니다.

**활용 예시**:

- 책 전체 분석 (여러 권 동시에)
- 대규모 코드베이스 리뷰
- 긴 비디오 콘텐츠 분석

**비교**:

| 모델 | 컨텍스트 | 대략적 분량 |
|------|----------|------------|
| Gemini 1.5 Pro | 2M 토큰 | 약 5,000페이지 |
| Claude 3.5 | 200K 토큰 | 약 500페이지 |
| GPT-4o | 128K 토큰 | 약 300페이지 |

### 2. Google Workspace 연동

Gmail, Docs, Drive와 직접 연동됩니다.

**가능한 작업**:

- Gmail에서 이메일 요약 및 답장 초안
- Google Docs에서 문서 작성 지원
- Google Sheets에서 데이터 분석
- Google Slides 프레젠테이션 생성

**사용 방법**:

1. Gemini Advanced 구독
2. Google Workspace에서 Gemini 활성화
3. 각 앱에서 Gemini 사이드패널 사용

### 3. Deep Research

심층 조사 후 종합 보고서를 자동 생성합니다.

**특징**:

- 여러 소스에서 정보 수집
- 체계적인 보고서 형식
- 출처 명시 및 인용

**예시 프롬프트**:

```
"2024년 전기차 시장 동향에 대해 심층 조사해줘.
주요 제조사별 판매량, 기술 트렌드, 향후 전망을 포함해줘."
```

### 4. 멀티모달 처리

텍스트, 이미지, 오디오, 비디오를 통합 처리합니다.

**가능한 작업**:

| 입력 | 가능한 작업 |
|------|------------|
| 이미지 | 분석, 설명, OCR, 비교 |
| 비디오 | 요약, 장면 설명, 자막 추출 |
| 오디오 | 전사, 요약, 번역 |
| PDF | 분석, 요약, 질의응답 |

### 5. 실시간 검색

Google 검색과 연동하여 최신 정보를 제공합니다.

```
프롬프트 예시:
"오늘 주요 뉴스를 검색해서 요약해줘"
"최신 애플 주가와 관련 뉴스를 알려줘"
```

### 6. Imagen (이미지 생성)

텍스트 설명으로 이미지를 생성합니다.

```
프롬프트 예시:
"한옥 카페에서 커피를 마시는 고양이,
수채화 스타일, 따뜻한 조명"
```
