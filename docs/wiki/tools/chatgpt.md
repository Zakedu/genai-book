## 핵심 기능

### 1. GPT-4o (Omni)

2024년 5월 출시된 최신 플래그십 모델입니다.

**특징**:

- 텍스트, 이미지, 오디오 통합 처리
- 이전 모델 대비 2배 빠른 속도
- 128K 컨텍스트 윈도우
- 실시간 음성 대화 지원

### 2. o1 (추론 모델)

2024년 9월 출시된 추론 특화 모델입니다.

**특징**:

- 복잡한 수학/과학 문제 해결
- 단계별 사고 과정 (Chain of Thought)
- 코딩 및 논리 문제에 강점
- 응답 시간이 길지만 정확도 높음

### 3. DALL-E 3

텍스트 설명으로 이미지를 생성하는 기능입니다.

```
프롬프트 예시:
"미래 도시의 스카이라인, 네온 조명이 반사되는 비 오는 밤,
사이버펑크 스타일, 4K 해상도"
```

### 4. 고급 데이터 분석 (Advanced Data Analysis)

Python 코드를 실행하여 데이터를 분석합니다.

**가능한 작업**:

- CSV, Excel 파일 분석
- 차트 및 그래프 생성
- 통계 분석 수행
- 데이터 변환 및 정제

### 5. GPTs (커스텀 챗봇)

특정 목적에 맞는 맞춤형 AI를 만들 수 있습니다.

**활용 예시**:

- 회사 내부 문서 기반 Q&A 봇
- 특정 주제 전문가 봇
- 업무 자동화 어시스턴트

### 6. Canvas

문서와 코드를 협업하여 편집하는 기능입니다.

**특징**:

- 실시간 편집 및 피드백
- 버전 히스토리 관리
- 코드 디버깅 지원

## 사용 팁

### 효과적인 프롬프트 작성

=== "기본 질문"
    ```
    파이썬으로 웹 크롤러 만들어줘
    ```

=== "상세 프롬프트"
    ```
    파이썬으로 웹 크롤러를 만들어주세요.

    요구사항:
    - requests와 BeautifulSoup 라이브러리 사용
    - 뉴스 사이트에서 제목과 링크 추출
    - 결과를 CSV 파일로 저장
    - 에러 처리 포함
    - 주석으로 코드 설명 추가
    ```

### 유용한 프롬프트 패턴

| 패턴 | 예시 |
|------|------|
| 역할 부여 | "당신은 경험 많은 Python 개발자입니다..." |
| 형식 지정 | "표 형식으로 정리해주세요" |
| 단계별 요청 | "먼저 개요를 작성하고, 그 다음 상세 내용을..." |
| 제약 조건 | "500자 이내로, 초등학생도 이해할 수 있게" |

### 웹 검색 활용

ChatGPT Plus에서는 실시간 웹 검색이 가능합니다.

```
프롬프트 예시:
"2024년 최신 아이폰 모델의 사양을 검색해서 알려줘"
```
