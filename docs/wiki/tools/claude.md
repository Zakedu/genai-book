## 모델 라인업

### Claude 3.5 Sonnet (추천)

가장 균형 잡힌 성능을 제공하는 주력 모델입니다.

**특징**:

- 200K 토큰 컨텍스트 (약 500페이지)
- 빠른 응답 속도
- 뛰어난 코딩 능력
- 자연스러운 한국어 지원

### Claude 3 Opus

최고 성능의 프리미엄 모델입니다.

**특징**:

- 가장 높은 추론 능력
- 복잡한 분석 작업에 적합
- 응답 시간이 상대적으로 김

### Claude 3.5 Haiku

빠르고 경제적인 경량 모델입니다.

**특징**:

- 가장 빠른 응답 속도
- 저렴한 API 비용
- 간단한 작업에 최적

## 핵심 기능

### 1. Projects (프로젝트)

문서를 업로드하여 지식 베이스를 구축할 수 있습니다.

**활용 예시**:

- 회사 매뉴얼 기반 Q&A 시스템
- 연구 논문 모음 분석
- 프로젝트 문서 정리 및 검색

**사용 방법**:

1. 프로젝트 생성
2. PDF, 텍스트 파일 업로드 (최대 5개)
3. 업로드된 문서 기반으로 대화

### 2. Artifacts (아티팩트)

코드, 문서, 다이어그램을 실시간으로 생성하고 미리봅니다.

**생성 가능한 유형**:

| 유형 | 설명 |
|------|------|
| HTML/CSS/JS | 웹페이지 실시간 미리보기 |
| React 컴포넌트 | 인터랙티브 UI 생성 |
| SVG 다이어그램 | 벡터 그래픽 생성 |
| Mermaid | 플로우차트, 시퀀스 다이어그램 |
| 마크다운 | 서식 있는 문서 |

**예시 프롬프트**:

```
React로 할 일 목록 앱을 만들어줘.
- 항목 추가/삭제 기능
- 완료 체크 기능
- 로컬 스토리지 저장
```

### 3. 긴 문서 처리

200K 토큰 컨텍스트로 장문 분석에 강점이 있습니다.

**활용 예시**:

- 100페이지 논문 요약
- 계약서 전문 분석
- 코드베이스 전체 리뷰

### 4. Constitutional AI

안전성을 고려한 응답을 생성합니다.

**특징**:

- 유해한 콘텐츠 생성 거부
- 불확실한 정보에 솔직함
- 편향 최소화 노력

## 사용 팁

### 효과적인 프롬프트 작성

=== "기본 요청"
    ```
    이 논문 요약해줘
    ```

=== "상세 프롬프트"
    ```
    다음 논문을 분석해주세요.

    분석 항목:
    1. 핵심 주장 (3줄 요약)
    2. 연구 방법론
    3. 주요 발견사항
    4. 한계점
    5. 실무 적용 가능성

    형식: 각 항목을 불릿 포인트로 정리
    ```

### Claude의 강점 활용

| 작업 유형 | 추천 프롬프트 패턴 |
|----------|-------------------|
| 글쓰기 | "~한 톤으로, ~한 독자를 대상으로 작성해줘" |
| 분석 | "여러 관점에서 분석하고, 장단점을 비교해줘" |
| 코딩 | "코드와 함께 설명을 주석으로 추가해줘" |
| 번역 | "자연스러운 한국어로, 의역보다 직역 위주로" |

### 멀티턴 대화 활용

Claude는 긴 대화 맥락을 잘 유지합니다.

```
1차: "마케팅 전략 보고서를 작성하려고 해"
2차: "타겟 고객층은 20-30대 직장인이야"
3차: "SNS 마케팅에 집중하고 싶어"
4차: "이제 보고서 초안을 작성해줘"
```
