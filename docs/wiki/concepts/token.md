# 토큰 (Token)

!!! info "최종 수정일: 2025-01-15"

**토큰**은 AI가 텍스트를 처리하는 기본 단위입니다.

---

## 정의

토큰은 단어, 단어의 일부, 또는 문장 부호가 될 수 있습니다. 영어에서는 대략 1 토큰 ≈ 4글자, 한국어에서는 1 토큰 ≈ 1-2글자 정도입니다.

---

## 왜 중요한가

- **비용 계산**: API 사용 시 토큰 수로 비용 산정
- **입력 제한**: 컨텍스트 윈도우는 토큰 수로 제한됨
- **응답 길이**: 출력 토큰 수 제한 가능
