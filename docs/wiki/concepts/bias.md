# 편향성 (Bias)

!!! info "최종 수정일: 2025-01-15"

**편향성**은 AI가 특정 집단이나 관점에 치우친 응답을 생성하는 현상입니다.

---

## 정의

AI 모델은 학습 데이터에 포함된 편향을 반영합니다. 이로 인해 성별, 인종, 문화 등에 대한 고정관념이 응답에 나타날 수 있습니다.

---

## 편향의 유형

| 유형 | 설명 |
|------|------|
| 성별 편향 | 특정 직업을 특정 성별과 연결 |
| 문화적 편향 | 서구 중심적 관점 |
| 확증 편향 | 기존 믿음을 강화하는 정보 선호 |
