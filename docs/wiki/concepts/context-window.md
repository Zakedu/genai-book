# 컨텍스트 윈도우 (Context Window)

!!! info "최종 수정일: 2025-01-15"

**컨텍스트 윈도우**는 AI가 한 번에 처리할 수 있는 토큰의 최대 개수입니다.

---

## 정의

대화의 "기억 용량"이라고 생각할 수 있습니다. 컨텍스트 윈도우를 초과하면 이전 내용을 "잊어버립니다".

---

## 주요 모델별 컨텍스트 윈도우

| 모델 | 컨텍스트 윈도우 |
|------|----------------|
| GPT-4 | 8K / 32K / 128K |
| Claude 3 | 200K |
| Gemini 1.5 | 1M+ |
