## 1. 책의 구조

이 책은 **5개의 파트**와 **부록**, 그리고 **위키 섹션**으로 구성되어 있습니다.

### 전체 구성도

**생성형 AI 완전 정복**

| Part 0 | Part 1 | Part 2 | Part 3 | Part 4 | Part 5 |
|--------|--------|--------|--------|--------|--------|
| 서론 | 이해 | 원리 | 활용 | 위험관리 | 미래 |

- **부록**: 템플릿, 용어, 참고자료
- **위키**: 개념, 도구, 사례연구

### 파트별 역할

| 파트 | 주제 | 핵심 질문 | 비중 |
|------|------|----------|------|
| **Part 0** | 서론 | 왜 배우는가? | 5% |
| **Part 1** | 이해 | AI란 무엇인가? | 15% |
| **Part 2** | 원리 | 어떻게 효과적으로 사용하는가? | 30% |
| **Part 3** | 활용 | 어디에 적용하는가? | 20% |
| **Part 4** | 위험관리 | 무엇을 주의해야 하는가? | 20% |
| **Part 5** | 미래 | 어떻게 준비하는가? | 10% |

## 2. 각 챕터의 표준 구조

모든 챕터는 일관된 구조로 작성되어 있습니다:

### 챕터 구성 요소

| 순서 | 구성 요소 | 설명 |
|------|----------|------|
| 1 | 학습 목표 (Learning Objectives) | 이 장을 마치면 ~할 수 있다 |
| 2 | 핵심 개념 (Core Concepts) | 이론적 설명 + 근거 자료 |
| 3 | 원리 설명 (Principles) | 왜 그런가? 메커니즘 해설 |
| 4 | 사례 분석 (Case Studies) | 성공/실패 사례 + 시사점 |
| 5 | 적용 지침 (Application Guidelines) | "이렇게 하면 좋다" 가이드라인 |
| 6 | 핵심 정리 (Key Takeaways) | 문제은행 출제 포인트 |
| 7 | 생각해볼 질문 (Discussion Questions) | 강의/토론용 질문 |

## 3. 독자 유형별 학습 경로

### 유형 1: 완전 입문자

> "생성형 AI가 뭔지 잘 모르겠어요"

```
Part 0 → Part 1 (전체) → Part 2 (Ch.3만) → Part 4 (Ch.9)
예상 소요 시간: 8-10시간
```

### 유형 2: 기초 사용자

> "ChatGPT는 써봤는데 제대로 활용을 못하겠어요"

```
Part 0 (0.2만) → Part 2 (전체) → Part 3 (관심 분야) → Part 4
예상 소요 시간: 12-15시간
```

### 유형 3: 중급 활용자

> "어느 정도 쓰고 있는데 체계적으로 정리하고 싶어요"

```
Part 2 (Ch.4-5) → Part 3 (전체) → Part 4 (전체) → Part 5
예상 소요 시간: 10-12시간
```

### 유형 4: 특정 목적 학습자

> "특정 분야에만 집중하고 싶어요"

=== "학생/연구자"

    ```
    Part 1 (Ch.1) → Part 2 (Ch.3-4) → Part 3 (Ch.6) → Part 4 (Ch.9-10)
    ```

=== "직장인/비즈니스"

    ```
    Part 2 (Ch.3, Ch.5) → Part 3 (Ch.7) → Part 4 (Ch.9, Ch.11)
    ```

=== "창작자/콘텐츠"

    ```
    Part 2 (Ch.3-4) → Part 3 (Ch.8) → Part 4 (Ch.10)
    ```

## 4. 학습 자료 활용법

### 교재 + 위키 + 문제은행 연계

**학습 자료 흐름**

1. **교재** (이론+원리) --> **위키** (최신정보) --> **문제은행** (역량검증)
2. 위 세 가지 자료가 **강의** (실습+토론)와 연결됨

### 각 자료의 역할

| 자료 | 역할 | 활용 시점 |
|------|------|----------|
| **교재** | 핵심 원리와 체계적 지식 | 첫 학습, 복습 |
| **위키** | 최신 정보, 상세 참조 | 심화 학습, 업데이트 확인 |
| **문제은행** | 이해도 점검, 역량 인증 | 학습 후 자가 평가 |
| **강의** | 실습, 토론, 질의응답 | 그룹 학습, 실무 적용 |

---

## 5. 효과적인 학습을 위한 팁

!!! tip "학습 효과를 높이는 5가지 방법"
    
    1. **순차적 학습**: Part 순서대로 학습하면 개념이 자연스럽게 쌓입니다
    2. **사례 집중**: 📌 표시된 사례 분석을 꼼꼼히 읽으세요
    3. **위키 연동**: 모르는 용어는 바로 위키에서 확인하세요
    4. **핵심 정리 활용**: 각 챕터 끝 '핵심 정리'는 시험 출제 포인트입니다
    5. **토론 질문 활용**: '생각해볼 질문'으로 깊이 있는 이해를 확인하세요

## 핵심 정리

<div class="key-concept">

**이 챕터의 핵심 포인트**

1. 책은 **서론 → 이해 → 원리 → 활용 → 위험관리 → 미래** 순서로 구성
2. 각 챕터는 **학습목표 → 개념 → 원리 → 사례 → 적용 → 정리 → 토론**의 표준 구조
3. **독자 유형에 따라** 선택적 학습 경로 가능
4. **교재-위키-문제은행**이 유기적으로 연결됨

</div>
