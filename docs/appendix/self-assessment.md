## Level 1: 기초 이해 (Chapter 1)

### AI 개념 이해

- [ ] 생성형 AI가 "다음 토큰 예측"으로 작동한다는 것을 설명할 수 있다
- [ ] AI, 머신러닝, 딥러닝, 생성형 AI의 포함 관계를 설명할 수 있다
- [ ] LLM(대규모 언어 모델)이 무엇인지 설명할 수 있다
- [ ] 환각(Hallucination)이 무엇인지, 왜 발생하는지 설명할 수 있다
- [ ] AI가 "사실을 아는 것"과 "그럴듯한 텍스트를 생성하는 것"의 차이를 이해한다

### 도구 이해 (Chapter 2)

- [ ] ChatGPT, Claude, Gemini의 주요 차이점을 설명할 수 있다
- [ ] 컨텍스트 윈도우가 무엇인지, 왜 중요한지 안다
- [ ] 토큰의 개념과 비용 계산 방식을 이해한다
- [ ] 각 도구의 특화 기능(예: ChatGPT의 GPTs, Claude의 Projects)을 안다
- [ ] 목적에 맞는 도구를 선택할 수 있다

---

## Level 2: 기본 활용 (Chapter 3-4)

### 프롬프트 기본기

- [ ] 5요소 프레임워크(맥락, 역할, 작업, 제약, 형식)를 적용할 수 있다
- [ ] 모호한 프롬프트를 명확한 프롬프트로 개선할 수 있다
- [ ] 역할 부여(Role-playing)를 효과적으로 활용할 수 있다
- [ ] 출력 형식을 원하는 대로 지정할 수 있다
- [ ] 시스템 프롬프트와 사용자 프롬프트의 차이를 이해한다

### 고급 프롬프팅

- [ ] Zero-shot과 Few-shot의 차이를 설명하고 적절히 선택할 수 있다
- [ ] Few-shot Learning을 위한 좋은 예시를 작성할 수 있다
- [ ] Chain of Thought(단계별 추론)를 적용할 수 있다
- [ ] 프롬프트 체이닝(연속적 프롬프트)을 활용할 수 있다

### 반복 개선 (Chapter 5)

- [ ] AI 응답을 평가하고 피드백을 제공할 수 있다
- [ ] Before/After 비교로 프롬프트를 반복 개선할 수 있다
- [ ] 디버깅 프롬프트를 사용해 문제를 파악할 수 있다
- [ ] 응답이 불만족스러울 때 다양한 접근법을 시도할 수 있다

---

## Level 3: 실전 활용 (Chapter 6-8)

### 학습/연구 활용 (Chapter 6)

- [ ] AI를 활용해 어려운 개념을 쉽게 설명받을 수 있다
- [ ] 논문이나 긴 문서를 효과적으로 요약할 수 있다
- [ ] AI로 연습 문제를 생성하고 풀이를 받을 수 있다
- [ ] AI 응답의 정확성을 외부 출처로 검증할 수 있다
- [ ] AI를 학습 도구로 활용하되 과도하게 의존하지 않는다

### 업무 활용 (Chapter 7)

- [ ] AI로 이메일, 보고서 초안을 효율적으로 작성할 수 있다
- [ ] 회의록 정리, 데이터 분석 등 업무에 AI를 활용할 수 있다
- [ ] 코딩 작업에 AI 어시스턴트를 활용할 수 있다
- [ ] 업무 기밀과 개인정보를 AI에 입력하면 안 되는 이유를 안다
- [ ] 업무 결과물에 대한 최종 책임이 사용자에게 있음을 이해한다

### 창작 활용 (Chapter 8)

- [ ] AI와 협업하여 글, 아이디어를 발전시킬 수 있다
- [ ] AI 생성 콘텐츠의 저작권 이슈를 이해한다
- [ ] AI를 도구로 활용하되 자신의 창작적 기여를 유지한다
- [ ] 이미지 생성 AI의 기본 사용법을 안다

---

## Level 4: 책임있는 활용 (Chapter 9-11)

### 환각 대응 (Chapter 9)

- [ ] 환각이 발생하기 쉬운 상황(최신 정보, 구체적 수치 등)을 식별한다
- [ ] AI 응답에서 환각 의심 신호를 인식할 수 있다
- [ ] 환각을 줄이는 프롬프트 기법을 적용할 수 있다
- [ ] 중요한 정보는 반드시 외부 출처로 검증한다
- [ ] Mata v. Avianca 사례 등 환각의 실제 위험성을 이해한다

### 윤리적 사용 (Chapter 10)

- [ ] AI 사용 시 저작권 문제를 인식하고 주의한다
- [ ] 학문적 진실성(표절, AI 사용 고지)의 중요성을 이해한다
- [ ] AI 편향의 원인과 유형을 설명할 수 있다
- [ ] AI 출력에서 편향을 인식하고 비판적으로 검토한다
- [ ] 개인정보와 기밀 정보를 AI에 입력하지 않는다

### 책임있는 사용 (Chapter 11)

- [ ] AI 사용 전 체크리스트를 적용할 수 있다
- [ ] 상황에 따라 AI 사용 여부를 판단할 수 있다
- [ ] AI 결과물에 대한 최종 책임을 인식하고 수용한다
- [ ] 필요한 경우 AI 사용을 적절히 표시한다

---

## Level 5: 전문가 수준 (Chapter 12-13)

### 핵심 역량 (Chapter 12)

- [ ] 비판적 사고를 유지하며 AI를 활용한다
- [ ] AI의 강점과 한계를 정확히 파악하고 있다
- [ ] 복잡한 작업을 위한 AI 워크플로우를 설계할 수 있다
- [ ] AI 도구 선택과 활용을 상황에 맞게 최적화할 수 있다
- [ ] 다른 사람에게 AI 활용법을 교육할 수 있다

### 미래 대비 (Chapter 13)

- [ ] 에이전트 AI의 개념과 가능성을 이해한다
- [ ] 멀티모달 AI의 발전 방향을 안다
- [ ] AI 관련 규제 동향을 이해한다
- [ ] 자신의 분야에서 AI 발전이 미칠 영향을 분석할 수 있다
- [ ] 지속적으로 AI 기술 발전을 학습하려는 자세를 가지고 있다
