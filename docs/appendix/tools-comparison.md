## ⚠️ 주의사항

### 공통 한계점

- **환각 현상**: 모든 AI가 잘못된 정보를 생성할 수 있음
- **최신 정보 부족**: 학습 데이터 기준일 이후 정보 부정확
- **개인정보 보호**: 민감한 정보 입력 시 주의 필요
- **저작권**: AI 생성 콘텐츠의 저작권 문제 인식 필요

### 서비스별 주의점

| 서비스 | 주의사항 |
|--------|----------|
| ChatGPT | 무료 버전 사용량 제한, 피크 시간 접속 지연 |
| Claude | 이미지 생성 불가, 실시간 정보 제한적 |
| Gemini | Google 계정 필수, 일부 국가 서비스 제한 |
