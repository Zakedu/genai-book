## ⚠️ 주의사항

### 공통 한계점

- **환각 현상**: 모든 AI가 잘못된 정보를 생성할 수 있음
- **최신 정보 부족**: 학습 데이터 기준일 이후 정보 부정확
- **개인정보 보호**: 민감한 정보 입력 시 주의 필요
- **저작권**: AI 생성 콘텐츠의 저작권 문제 인식 필요

### 서비스별 주의점

| 서비스 | 주의사항 |
|--------|----------|
| ChatGPT | 무료 버전 사용량 제한, 피크 시간 접속 지연 |
| Claude | 이미지 생성 불가, 실시간 정보 제한적 |
| Gemini | Google 계정 필수, 일부 국가 서비스 제한 |

### Perplexity AI

**특징:** 실시간 웹 검색에 특화된 AI 검색 엔진

| 항목 | 내용 |
|------|------|
| **강점** | 실시간 정보 검색, 출처 명시, 팩트 중심 |
| **약점** | 창작/코딩에는 다른 도구 대비 약함 |
| **무료 버전** | 일일 검색 제한, 기본 모델 사용 |
| **Pro 버전** | $20/월, GPT-4/Claude 선택 가능 |
| **추천 용도** | 리서치, 최신 정보 검색, 팩트체킹 |

```
특화 기능:
- 답변에 출처 URL 자동 표시
- "Focus" 기능으로 검색 범위 지정 (학술, 뉴스, 유튜브 등)
- 후속 질문 자동 제안
```
